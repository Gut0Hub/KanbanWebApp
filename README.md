📌 Projeto Integrador - Sistema de Gerenciamento de Tarefas

🏢 Organização Parceira

Este projeto foi desenvolvido com base na empresa **MaximaTech**, que atua no ramo de tecnologia com soluções voltadas para pedidos e suporte técnico.

---

🎯 Objetivo

Desenvolver um sistema web de gerenciamento de tarefas com foco no controle e acompanhamento de chamados técnicos, proporcionando maior organização e eficiência no atendimento.

---

🚨 Problema Identificado

Durante a análise do ambiente de suporte técnico, foram identificadas dificuldades como:

* Falta de controle eficiente dos chamados
* Dificuldade no acompanhamento de status
* Falta de organização das tarefas da equipe
* Dependência de outros setores para resolução

---

💡 Solução Proposta

Foi proposto o desenvolvimento de um sistema web com funcionalidades de gerenciamento de tarefas, utilizando o modelo Kanban para organização dos chamados.

---

⚙️ Metodologia

O projeto foi desenvolvido com base em uma abordagem ágil, utilizando ferramentas de apoio para organização e prototipação:

* Trello (gestão de tarefas)
* Figma (prototipação das telas)

---

💻 Funcionalidades do Sistema

O sistema desenvolvido contempla as seguintes funcionalidades:

* 🔐 Login de usuário
* 📋 Criação de chamados
* 📊 Visualização em quadro Kanban
* 🔄 Organização por status:

  * Pendente
  * Em atendimento
  * Resolvido
